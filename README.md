## Python Project

### Commands

**Install Django**<br />
```
python -m pip install django
```

**Start Project**<br />
```
python -m django startproject PROJECT_NAME
```

**Create/Start Application**<br />
```
python manage.py startapp APP_NAME
```

**Run Server**<br />
```
python manage.py runserver
```
Ctrl+C or Ctrl+Break to quit server.
